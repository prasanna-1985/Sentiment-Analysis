# Sentiment Analysis on Amazon Reviews
Amazon is an e-commerce site and many users provide review comments on this online site. This project focuses on sentiment analysis of Amazon customer reviews for amazon products. Many customers chose to provide their experience, opinion, feedback etc. Such product reviews are rich in information consisting of feedback shared by users.
